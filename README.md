# Peter Strössler – Personal Landing Page

## Files

- **index.html** – Main page markup. Includes a dark‑theme hero section (title, subtitle, intro, CTA button), an about section, and a contact form.
- **css/style.css** – Global stylesheet implementing the dark theme, primary blue (`#4A6CF7`) and orange accent (`#FF8800`). Handles responsive layout.
- **js/main.js** – Vanilla JavaScript for:
  - Scrolling to the about section when the CTA button is clicked.
  - Basic validation of the contact form and showing a success/error message.
- **assets** – The hero image and logo are referenced directly from the existing draft folder (`draft/src/images/hero/hero-image.webp` and `draft/src/images/logo/logo.svg`). No new copies are needed.

The page works without any external frameworks – just HTML, CSS and vanilla JS.
